# ANHANG A · Glossar

| Begriff | Definition |
| :--- | :--- |
| **AEO** | Answer Engine Optimization — Optimierung für direkte Antworten in KI-Systemen und Featured Snippets. |
| **Agentic AI** | KI-Systeme, die autonom planen, entscheiden und handeln in mehrstufigen Workflows. |
| **Chain-of-Thought** | Prompting-Technik, die KI zur Schritt-für-Schritt-Begründung auffordert — reduziert Halluzinationen. |
| **EU AI Act Art. 4** | Pflicht zur AI Literacy für alle MitarbeiterInnen, die KI einsetzen — seit 02.02.2025. |
| **Few-Shot Prompting** | 2–3 Beispiele im Prompt, die der KI zeigen, was exzellenter Output bedeutet. |
| **GEO** | Generative Engine Optimization — Optimierung für Sichtbarkeit in KI-generierten Antworten. |
| **LLMO** | Large Language Model Optimization — Optimierung für Relevanz in LLM-Trainingsdaten und -Outputs. |
| **Privacy by Design** | Datenschutz als architektonisches Prinzip (Art. 25 DSGVO), nicht als Nachgedanke. |
| **Zero-Party Data** | Vom Nutzer aktiv und bewusst bereitgestellte Daten — höchste Qualität, vollständig DSGVO-konform. |
| **Zero-Click Search** | Suchanfragen, die ohne Klick auf eine Website beantwortet werden — ca. 70% Anteil ab 2026. |
| **C2PA** | Coalition for Content Provenance and Authenticity — Standard zur manipulationssicheren Herkunftskennzeichnung digitaler Medien. |
