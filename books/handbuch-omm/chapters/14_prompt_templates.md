# ANHANG B · Prompt-Template-Bibliothek (Starter Set)

**Template 1 — Strategisches Briefing erstellen:**
```
Du bist [PERSONA]. Wir arbeiten an [PROJEKT]. Zielgruppe: [ZG]. Ziel: [ZIEL].
Erstelle ein strategisches Briefingdokument mit: Ziel, ZG-Analyse, Kernbotschaft, Kanalstrategie, KPIs, Risiken.
Format: Markdown. Constraints: DSGVO-konform, EU AI Act-konform, max [X] Wörter.
RECAP: [2–3 kritische Punkte].
```

**Template 2 — Gegenkontrolle:**
```
Du hast dieses Dokument erstellt: [DOKUMENT]. Jetzt bist du ein kritischer Senior-Reviewer.
Finde: logische Widersprüche, fehlende Informationen, Compliance-Risiken, unrealistische Annahmen.
Sei gnadenlos. Liste jeden Punkt einzeln.
```

**Template 3 — Brand Voice Content:**
```
[BRAND VOICE DOCUMENT einfügen].
Auf Basis dieser Brand Voice: Erstelle [CONTENT-FORMAT] über [THEMA] für [ZIELGRUPPE].
Kanal: [KANAL]. Ziel: [ZIEL]. Format: [FORMAT]. Länge: [LÄNGE].
```

**Template 4 — Devil's Advocate:**
```
Hier ist meine Strategie / mein Concept: [EINGABE].
Kritisiere es wie ein erfahrener Wettbewerber, der jeden Schwachpunkt ausnutzen würde.
Dann: Was sind die drei größten Risiken? Was würde ein Experte anders machen?
```

**Template 5 — Retrospektive mit KI:**
```
Hier ist der vollständige Output unserer Session: [OUTPUT].
Bewerte ihn auf einer Skala von 1–10 in: Klarheit, Strategie, Umsetzbarkeit, Compliance, Originalität.
Erkläre jeden Score. Was hätte einen Score von 10 ergeben?
```
