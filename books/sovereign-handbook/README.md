# SOVEREIGN – Developer Handbook — Monorepo Entry
Dieses Verzeichnis enthält die modularen Kapitel des SOVEREIGN Developer Handbooks.

## Kompilierung
Führe das Python-Skript im `build/` Verzeichnis aus, um alle Kapitel in ein einziges Dokument zusammenzuführen:
```bash
python build/build_book.py
```
Das fertige Dokument wird nach `dist/compiled_book.md` geschrieben.
